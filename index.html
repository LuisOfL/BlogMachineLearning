Introducción al Machine Learning

Descubre cómo las máquinas pueden aprender de los datos y hacer predicciones automáticamente

¿Qué es el Machine Learning?

¿Alguna vez te has preguntado cómo las empresas pueden procesar tantos datos?

Imagina que quieres hacer una encuesta para saber cuántas horas estudian los estudiantes y su GPA, y de repente tienes 10 millones de respuestas. ¿Cómo contarías tantos datos?

¿Pedirías ayuda a 100 personas? Eso no tendría sentido en estos tiempos. Aquí es donde entra el machine learning.

Machine Learning (ML) es un tipo de Inteligencia Artificial (IA) que le da a las computadoras la capacidad de aprender de los datos sin estar programadas explícitamente para cada tarea específica.

En términos más simples, tu computadora hará el trabajo por ti. No tendrás que contar las 10 millones de respuestas una por una, e incluso podrás predecir, por ejemplo, qué calificación obtendría si estudio 6 horas.

Esto suena genial, ¿verdad? Lo que antes requería varias personas ahora puede hacerlo una sola persona.

Analicemos el ejemplo dado:

Instrucciones: Dados 10 millones de puntos de datos con horas estudiadas y una calificación, predice qué obtendré si estudio 4 horas. Supón que las 10 millones de personas respondieron ambos campos, por lo que no hay valores nulos. Podría haber casos donde los datos estén incompletos o donde Panchito puso 90 para la calificación y Fulano puso 9.0, pero para simplificar, asumamos que los datos son uniformes.

¿Qué debemos hacer?

Sabemos que mientras más estudies, mejores calificaciones obtienes. Depende de la calidad del estudio, pero en general funciona así. Entonces sabemos que los datos tienen una tendencia fácil de encontrar. Para tener una mejor referencia, visualicemos los datos en un plano, asumiendo para este caso 10 puntos de datos, donde las horas estudiadas son x y la calificación es y.

Visualización de Datos

Ahora, después de mirar el gráfico, está más claro: se forma una línea que sube a medida que avanza. Esto se puede resumir como: "si estudio más horas, obtendré mejores calificaciones".

Selección del Modelo

Como se mencionó antes, existen varios modelos de machine learning. ¿Cuál deberíamos usar?

"Quiero saber mi calificación si estudio 6 horas." Entonces, conozco las horas de estudio, pero no sé la calificación; eso es lo que quiero averiguar.

¿Qué tipo es la calificación? ¿Es un valor numérico? ¿Es una variable continua o discreta?

  • Variable continua: Incluye números con puntos decimales.
  • Variable discreta: Solo números enteros.

Entonces, sé que la calificación es un valor numérico y es una variable discreta. El modelo que mejor se adapta a mis necesidades es: Regresión Lineal.

Regresión Lineal

El primer modelo que vamos a ver es la regresión lineal. Es más fácil de lo que parece, si entiendes primero la parte matemática y luego la parte algorítmica.

¿Qué es la Regresión Lineal?

Usemos el gráfico que vimos anteriormente. Si te fijas, podemos dibujar una línea que se ajuste a la mayoría de los puntos de datos.

De los siguientes gráficos, dibuja mentalmente la línea que mejor se ajuste a los datos.

Encuentra la Línea de Mejor Ajuste

Fácil, ¿verdad? Eso es regresión lineal, nada del otro mundo. El pequeño problema es encontrar esa línea matemáticamente y luego programarla.

Hay varias formas de hacerlo, desde métodos numéricos como el descenso de gradiente que involucra derivadas y es más óptimo para grandes datos, hasta un método muy fácil llamado mínimos cuadrados que es álgebra pura.

Fórmulas de Regresión Lineal

y = a + bx
b = Σ[(xᵢ - x̄)(yᵢ - ȳ)] / Σ[(xᵢ - x̄)²]
a = ȳ - bx̄

¿Qué significa cada cosa?

Tenemos 3 fórmulas. Estoy seguro de que ya sabes qué es la primera - así es, es la fórmula punto-pendiente de una línea. Es obvio, ¿verdad? Queremos encontrar una línea. b es la pendiente, que hace girar la línea, por así decirlo. La dirección hacia donde apunta la línea depende de la pendiente, y a es la intersección con el eje y. Con estos parámetros, podemos dibujar infinitas líneas, y una de ellas debe ser la nuestra.

Ya sabemos que con pendiente m e intersección a, podemos tener infinitas líneas, y una de ellas debe ser la que estamos buscando.

Si miramos, ya tenemos la fórmula para b y a, así que solo necesitamos hacer operaciones básicas y voilà, tendremos nuestra línea que se ajusta a nuestros datos.

¿Cómo empezarías?

Si te fijas, para a, necesitas b. Así que simplemente calculas b primero, luego a, y finalmente tu ecuación de línea.

Sabemos que x̄: es la media de x, donde x es un conjunto de datos.

Por ejemplo, si tenemos 3,5,2,1 → la media o promedio es (3+5+2+1)/4. ¿Por qué 4? Porque es el número de elementos en mi conjunto de datos.

Ejemplo de Regresión Lineal

Supongamos que tienes una lista:

  • x (horas estudiadas) = [3,5,2,4,6,2]
  • y (calificación obtenida) = [7,9,6,8,10,5]

Para entender la regresión lineal, primero trabajaremos con conjuntos de datos pequeños. Lo primero que necesitas calcular es b. Para hacer esto, calculamos la media de x e y:

  • x(media) = (3+5+2+4+6+2)/6 = 22/6 ≈ 3.6667
  • y(media) = (7+9+6+8+10+5)/6 = 45/6 = 7.5

Paso 2: Calcular el numerador

Luego, para cada valor en la lista x, restamos la media y lo multiplicamos por el valor correspondiente en y menos la media de y:

(3-3.6667)(7-7.5) + (5-3.6667)(9-7.5) + (2-3.6667)(6-7.5) + (4-3.6667)(8-7.5) + (6-3.6667)(10-7.5) + (2-3.6667)(5-7.5) ≈ 15

Paso 3: Calcular el denominador

Para cada valor en x, resta la media, elévalo al cuadrado y súmalos:

(3-3.6667)² + (5-3.6667)² + (2-3.6667)² + (4-3.6667)² + (6-3.6667)² + (2-3.6667)² ≈ 13.3333

Paso 4: Calcular la pendiente b

Divide el numerador por el denominador: b = 15 / 13.3333 ≈ 1.125

Paso 5: Calcular la intersección a

a = y(media) - b * x(media) = 7.5 - (1.125 * 3.6667) ≈ 3.375

Paso 6: Ecuación de la línea de regresión

y = bx + a → y = 1.125x + 3.375

Esta es la línea óptima que se ajusta a estos datos, y nos permite predecir la calificación basada en las horas de estudio.

Predicción de Calificación

Usando la ecuación de la línea de regresión que obtuvimos:

y = 1.125x + 3.375

Si queremos predecir la calificación para 6 horas de estudio:

y = 1.125 * 6 + 3.375 = 10.125 → aproximadamente 10 puntos

Implementación en Python:

Algoritmo de Regresión Lineal en Python


# Datos
x = [3, 5, 2, 4, 6, 2]  # Horas estudiadas
y = [7, 9, 6, 8, 10, 5]  # Calificación obtenida

# Función para calcular la media
def mean(lst):
    return sum(lst) / len(lst)

# Calcular medias
x_mean = mean(x)
y_mean = mean(y)

# Calcular numerador y denominador para b
numerator = sum((xi - x_mean) * (yi - y_mean) for xi, yi in zip(x, y))
denominator = sum((xi - x_mean)**2 for xi in x)

# Calcular pendiente b e intersección a
b = numerator / denominator
a = y_mean - b * x_mean

# Función para predecir calificación basada en horas estudiadas
def predict(hours):
    return b * hours + a

# Predicción para 6 horas de estudio
predicted_grade = predict(6)
print(f"Predicción: {predicted_grade}")
